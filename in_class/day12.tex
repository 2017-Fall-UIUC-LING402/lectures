Read about the Caesar cipher:
\url{https://en.wikipedia.org/wiki/Caesar_cipher}

\ \\

Implement a program that can encode or decode a string using the Caesar cipher.  
Decoding could be done using a known shift value or using brute force.

\ \\ 

Only shift through asciii upper- and lowercase letters.  
The case of each letter should be retained.
Do not encode punctuation and whitespace.

\ \\

So for example, 'Hello World!' becomes 'Jgnnq Yqtnf!' with a shift of 2.
