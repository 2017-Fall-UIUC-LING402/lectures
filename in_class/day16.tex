\frametitle{Play time}

View the rules for solving the \textbf{Towers of Hanoi} problem:
\url{https://en.wikipedia.org/wiki/Tower_of_Hanoi}

\ \\

Implement a class called \texttt{Towers} with 3 poles, \texttt{start, middle, goal}, 
and whose constructor takes an 
integer representing the number of disks placed on the \texttt{start} pole.

\ \\

Implement a program that prompts the user for the number of disks (an int), 
instantiates a \texttt{Towers} object with that many disks on the \texttt{start} pole,
and then successfully moves all disks to the \texttt{goal} pole
using a recursive algorithm that follows the rules for solving the Towers of Hanoi.
